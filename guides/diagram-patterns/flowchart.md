# 플로우차트

의사결정 트리.

**언제**: 알고리즘 선택 가이드, 디버깅 분기, 제어 흐름

## 구성

| 요소 | 크기 | 색 |
|---|---|---|
| 시작 박스 | 110 × 32, `rx=16` (타원형) | `#f5f5f5` (중립) |
| 결정 박스 | 200 × 40, `rx=4` | `#e8f4fa` fill + `#5db0d7` stroke |
| 결과 박스 | 160 × 40, `rx=4` | `#fff7e6` fill + `#ff5544` stroke |

## 분기 방향 — 일관성

```
Yes → 우측 (결과)     라벨 10px, #5db0d7
No  → 아래  (다음 결정) 라벨 10px, #888
```

**방향을 섞지 않는다.** 모든 No의 종착지가 가운데 아래 기본 결과다.

## 주의

- 결정 박스 텍스트는 **짧은 질문형 한 줄**
- 결과 박스도 한 줄
- 4단계 결정 트리 캔버스: 720 × 460 정도
