# 그래프

정점·간선.

**언제**: 그래프 표현, BFS/DFS, 위상정렬, 최단경로, MST, Union-Find

## 기본

- 원 노드 + 직선 간선 (무방향)
- 방향은 polygon 화살촉 (→ `arrowhead.md`)
- 가중치는 간선 중앙 **흰 박스** 라벨
- 간선은 노드 테두리 **바깥**에서 시작/끝

## 예제 통일

**다이어그램·본문 코드·캡션이 전부 같은 예제를 써야 한다.**

```python
{0:[1,2], 1:[0,3,4], 2:[0,4], 3:[1], 4:[1,2,5], 5:[4]}

BFS: 0 → 1 → 2 → 3 → 4 → 5
DFS: 0 → 1 → 3 → 4 → 2 → 5
```

방문 순서는 **인접 리스트 저장 순서에 의존한다.** 예제가 다르면 순서가 달라진다.

## 패턴

### 거리 레인 (BFS)
거리 레벨을 **가로 띠**로 쌓는다. 같은 거리 정점은 같은 줄, 간선은 위→아래.

> **동심원 배치는 폐기했다.** 간선이 사방으로 뻗고 라벨이 떠서 가독성이 나쁘다.

### 한 줄기 + 백트래킹 (DFS)
forward 간선(주황 실선) + backtrack(회색 점선) + 방문 순서 뱃지(노드 우상단 작은 원).
DFS 트리 깊이로 노드를 배치한다.

### 큐/스택 시퀀스
좌: 그래프 / 우: 단계별 상태표.
**첫 행은 "시작: N 넣음"부터.** DFS 스택은 중복 push + 꺼낼 때 검사(건너뜀)까지 표현.

### 연결 요소 분리
점선 박스로 컴포넌트를 묶고 `①②③` 재시작 순서 라벨. 컴포넌트별 색 구분.

### 거리 배열 / 행렬
완화 단계별 표. 갱신 칸은 빨강.
표기: `"정점: 옛값 → 새값"` **한 형식으로 통일**.

### 다단계 패널
패널 폭 × 개수가 viewBox를 넘지 않게 노드 레이아웃을 압축.
패널 전환 화살표는 노드와 겹치지 않는 높이(하단)에.

## 주의

- **폭은 콘텐츠에 맞춘다.** 좌 그래프 + 우 표 구조는 800 전후. 과한 여백 금지
- 표가 N열이면 셀 폭을 줄여 viewBox 안에 맞춘다 (5열 잘림 사례 있음)
- 밀집 그래프 간선 수는 `V(V-1)/2`. **`V²`가 아니다** (점근만 Θ(V²))
