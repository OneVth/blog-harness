# 트리

**언제**: 힙, BST, 레드-블랙, B-트리, 트라이, 분할 정복

## 노드

| 항목 | 값 |
|---|---|
| 반지름 | 22 (2자리 숫자까지 가독) |
| 폰트 | 14px |
| 레벨 간 y 간격 | 50px |
| 노드 가로 간격 | 중심 거리 80px |

**4레벨(15노드)까지 폭 720 안에 들어간다.** 그 이상은 반지름 축소 또는 폭 확장.

## 작성 순서

```
1. 연결선 <line>   ← 먼저 (노드가 덮는다)
2. 노드 <circle>
3. 노드 안 텍스트
4. 노드 옆/아래 라벨
```

**간선은 노드 반지름만큼 떨어진 지점에서 시작/끝낸다.** 원에 겹치면 안 된다.
노드 반지름이 다르면(강조 노드 r=44 등) **화살촉 좌표를 그 반지름으로 재계산**한다.

## 좌표 (3레벨 기준)

```
Level 0 (root):   x=360 (가운데)
Level 1 (2개):    x=280, 440       (root ±80)
Level 2 (4개):    x=240, 320, 400, 480   (간격 80)
```

## 특수 노드

| 종류 | 표현 |
|---|---|
| **다중 키** (B-트리) | 셀로 분할된 사각형. 키 사이 구간(k+1개)에서 자식이 갈라짐 |
| **색 노드** (RBT) | RED = 연한 채움(`#ffe4e0`/stroke `#ff5544`) / BLACK = 진한 채움(`#4a4a4a`/흰 글씨). **범례 동반** |
| **이중 검정** (RBT 삭제) | 이중 테두리 (외곽 `#ff5544` 굵게 + 내부 얇은 원) |
| **단어 끝** (트라이) | 이중 원 (내부에 작은 원) |

## 변형

### before → after (회전·분할)
좌우 2-state 또는 3-state(이중 회전). `①②③` 단계 주석.
**중위 순회 보존 같은 불변식을 캡션에 명시한다.**

### 압축 트라이
간선 라벨에 문자 하나가 아니라 **문자열 조각**("te", "st").
일반 트라이(글자마다 노드) vs 압축(분기점만) 대비.
