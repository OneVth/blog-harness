# 집합 분류

포함 관계 (A ⊂ B).

**언제**: URL ⊂ URI, www ⊂ naver.com, Host = End-point + Switch, 자료구조 taxonomy

## 구성

- 큰 타원 (상위 개념) 안에 사각형 부분집합 둘 이상
- 또는 중첩 박스 (트리 ⊃ 이진 트리 ⊃ BST)
- 상하 분류로 taxonomy (선형 / 비선형)

## 주의 — 검산

내부 블록이 타원을 넘지 않게:

```
((x - cx) / rx)² + ((y - cy) / ry)² < 1     ← 타원 안
```

타원 rx/ry를 블록보다 **충분히 크게** 잡는다.

**라벨은 타원 밖에 배치한다.** 안에 넣으면 블록과 겹친다.

## 변형

### 스펙트럼
두 축의 반비례 관계를 축 위에 배치. (보안성 ↔ P2P 가능성)

### 매트릭스
구조 × 계층 격자에 항목 배치. 글 전체의 지도 역할.
